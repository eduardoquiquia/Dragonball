# Laboratorio 13 - Conocer las Principales Bibliotecas Frontend

## Descripción

En este laboratorio se exploraron algunas de las principales bibliotecas utilizadas en el desarrollo Frontend moderno. Se trabajó sobre un proyecto React creado con Vite, incorporando herramientas ampliamente utilizadas para la construcción de interfaces de usuario modernas, reutilizables y responsivas.

Durante el desarrollo se integraron tecnologías como Tailwind CSS para el estilado basado en utilidades y shadcn/ui para la implementación de componentes reutilizables. Como actividad práctica se desarrolló una Landing Page temática para validar el funcionamiento de las herramientas instaladas.

---

## Objetivos

- Conocer las principales bibliotecas utilizadas en el ecosistema Frontend.
- Comprender el proceso de configuración de un proyecto React moderno.
- Integrar Tailwind CSS para la construcción de interfaces.
- Utilizar componentes reutilizables mediante shadcn/ui.
- Aplicar buenas prácticas en la organización de proyectos Frontend.

---

## Tecnologías Utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

# Ejecución del Proyecto

## Instalar dependencias

```bash
npm install
```

## Ejecutar en modo desarrollo

```bash
npm run dev
```

## Compilar para producción

```bash
npm run build
```

## Vista previa de producción

